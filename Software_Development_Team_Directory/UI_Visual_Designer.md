# UI/Visual Designer

## Role
The UI (User Interface) Designer focuses on the product's visual appearance and interactive elements. They ensure the product is beautiful, on-brand, and visually intuitive.

## Responsibilities
- **High-Fidelity Mockups:** Creating the final look and feel of every screen.
- **Design Systems:** Building and maintaining a library of reusable components (buttons, fonts, colors).
- **Brand Consistency:** Ensuring the product aligns with the company's visual identity.
- **Visual Hierarchy:** Using color, typography, and spacing to guide the user's eye to important actions.

## Tools and Technologies
- **Design Core:** Figma (industry standard), Sketch, Adobe XD.
- **Design Systems Tooling:** Storybook (for alignment with code), Zeroheight.
- **Visual Assets:** Adobe Creative Cloud (Photoshop, Illustrator).
- **Handover:** Zeplin, Figma Dev Mode.
