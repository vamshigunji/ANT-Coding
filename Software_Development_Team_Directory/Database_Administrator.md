# Database Administrator (DBA)

## Role
The DBA is the guardian of the company's most valuable asset: its data. They focus on the physical storage, performance, and security of the database engines themselves. In modern teams, this is often a "Cloud Database Engineer."

## Responsibilities
- **Performance Tuning:** Optimizing slow queries, managing indexes, and identifying bottlenecks.
- **Backups & Recovery:** Ensuring that no data is ever lost and that the system can recover from a catastrophic failure.
- **Replication & Sharding:** Designing how data is distributed across multiple servers for high availability and scale.
- **Security:** Managing data encryption at rest and in transit, and auditing database access.

## Tools and Technologies
- **Engines:** PostgreSQL, MySQL, SQL Server, Oracle.
- **Cloud Managed Services:** AWS RDS, Google Cloud SQL, Azure SQL.
- **Monitoring:** SolarWinds, Datadog Database Monitoring, pgAdmin.
- **Automation:** Liquibase, Flyway (for database migrations).
