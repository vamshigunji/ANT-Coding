# Security Engineer (DevSecOps)

## Role
Security Engineers protect the company's data and infrastructure from cyber threats. They embed security checks into every stage of the development process (Shift-Left security).

## Responsibilities
- **Vulnerability Scanning:** Automatically checking code for security flaws (SAST/DAST).
- **Identity & Access:** Managing who has access to which systems (IAM).
- **Compliance:** Ensuring the product meets legal standards like GDPR or HIPAA.
- **Incident Response:** Reacting to security breaches or suspicious activity.

## Tools and Technologies
- **Scanning:** Snyk, SonarQube, Wiz, Checkmarx.
- **IAM:** AWS IAM, Okta, Auth0.
- **Secrets Management:** HashiCorp Vault, AWS Secrets Manager.
- **Network Security:** Cloudflare, Palo Alto Networks.
- **Pentesting:** Burp Suite, Metasploit.
