# Technical Support Engineer

## Role
Technical Support Engineers are the "Tier 2/3" support team. They handle complex issues that standard customer service can't solve. They often have deep technical knowledge and act as a filter between customers and the engineering team.

## Responsibilities
- **Bug Triaging:** Investigating reported issues and determining if they are true bugs or user error.
- **Log Investigation:** Digging into server logs to find the root cause of a specific user's failure.
- **Internal Tooling:** Building scripts or dashboards to help support teams help customers faster.
- **Technical Documentation:** Writing help articles for other developers or power users.

## Tools and Technologies
- **Ticketing:** Zendesk, Intercom, Jira Service Management.
- **Log Search:** Splunk, LogQL (Grafana Loki), Datadog.
- **Database Access:** SQL (for investigating user data records).
- **Communication:** Slack, Zoom.
