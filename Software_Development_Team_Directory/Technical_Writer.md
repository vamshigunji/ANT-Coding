# Technical Writer

## Role
Technical Writers translate complex technical information into clear, usable documentation for developers (External Docs) or internal employees (Internal Knowledge Base).

## Responsibilities
- **API Documentation:** Writing guides for developers to use the company's APIs.
- **Tutorials & SDK Guides:** Creating step-by-step instructions for getting started with the product.
- **Release Notes:** Clearly explaining what has changed in each new version of the software.
- **Documentation Strategy:** Organizing the structure of the help center or developer portal.

## Tools and Technologies
- **Static Site Generators:** Docusaurus, MkDocs, Hugo.
- **Documentation Platforms:** GitBook, ReadMe.io, Stoplight.
- **Docs-as-Code:** GitHub, Markdown, AsciiDoc.
- **Screenshot/Video:** Snagit, CleanShot X, Loom.
