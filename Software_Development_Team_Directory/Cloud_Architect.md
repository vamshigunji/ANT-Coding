# Cloud Architect

## Role
The Cloud Architect designs the company's virtual data center. They are experts in cloud providers (AWS, GCP, Azure) and focus on making sure the infrastructure is cost-efficient, globally redundant, and highly secure.

## Responsibilities
- **Region Strategy:** Deciding where in the world data should be stored for speed and legal compliance.
- **Cost Optimization:** Reducing the monthly cloud bill by choosing the right instance types and storage tiers.
- **Network Design:** Setting up VPCs (Virtual Private Clouds), VPNs, and firewalls.
- **Disaster Recovery:** Planning how to bring the system back online if an entire cloud region goes down.

## Tools and Technologies
- **Cloud Providers:** AWS (Amazon Web Services), GCP (Google Cloud Platform), Microsoft Azure.
- **Cost Management:** AWS Cost Explorer, Vantage, Kubecost.
- **Network:** Cloudflare, AWS Route 53.
- **Compliance:** Drata, Vanta (to ensure the cloud setup is SOC2 compliant).
