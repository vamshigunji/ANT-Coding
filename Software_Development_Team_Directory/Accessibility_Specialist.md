# Accessibility (a11y) Specialist

## Role
The Accessibility Specialist ensures that the software is usable by as many people as possible, including those with visual, auditory, motor, or cognitive disabilities. They ensure compliance with laws like the ADA and standards like WCAG.

## Responsibilities
- **Auditing:** Reviewing designs and code for accessibility barriers.
- **Screen Reader Testing:** Ensuring the application is fully functional using VoiceOver, NVDA, or JAWS.
- **Semantic HTML Advocacy:** Training engineers to use proper HTML elements to ensure compatibility with assistive tech.
- **Color Contrast & Focus:** Ensuring the UI is readable and navigable without a mouse.

## Tools and Technologies
- **Audit Tools:** Axe DevTools, Wave, Lighthouse.
- **Assistive Tech:** VoiceOver (iOS/macOS), TalkBack (Android), NVDA (Windows).
- **Design Checkers:** Stark, Figma A11y plugins.
