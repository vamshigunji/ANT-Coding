# Analytics Engineer

## Role
Analytics Engineers provide the bridge between Data Engineers and Business Analysts. They take the raw data in the warehouse and transform it into clean, well-modeled tables that are ready for the business to use.

## Responsibilities
- **Data Modeling:** Writing modular, efficient SQL to clean and join disparate data sources.
- **Metric Definition:** Ensuring that "Revenue" or "User Count" is calculated the same way across the whole company.
- **Performance:** Optimizing data warehouse queries for faster reporting.
- **Quality Assurance:** Writing tests to ensure the data in reports is accurate.

## Tools and Technologies
- **Transformation Core:** dbt (industry standard).
- **SQL Mastery:** Advanced SQL on Snowflake, BigQuery, or Redshift.
- **BI Tools:** Looker (LookML), Tableau, Metabase.
- **Version Control:** Git (applying engineering best practices to data).
