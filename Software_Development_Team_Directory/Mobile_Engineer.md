# Mobile Engineer

## Role
Mobile Engineers specialize in building applications for smartphones and tablets. They must account for specific hardware constraints (battery life, varied screen sizes) and platform-specific guidelines (App Store/Play Store rules).

## Responsibilities
- **Native Development:** Writing code specifically for iOS (Swift) or Android (Kotlin).
- **Cross-Platform Development:** Using frameworks that target both platforms with one codebase.
- **Offline Sync:** Handling data synchronization when the user has poor internet connectivity.
- **Push Notifications:** Managing real-time engagement via device alerts.

## Tools and Technologies
- **iOS:** Swift, SwiftUI, Xcode.
- **Android:** Kotlin, Jetpack Compose, Android Studio.
- **Cross-Platform:** React Native, Flutter, Expo.
- **Backend as a Service (BaaS):** Firebase, Supabase.
- **Testing/Distribution:** TestFlight, App Center, BrowserStack.
