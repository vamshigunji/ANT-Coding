# UX Writer / Content Designer

## Role
The UX Writer is responsible for every word inside the product. They ensure the language is clear, concise, and helps the user complete their task.

## Responsibilities
- **Microcopy:** Writing labels, buttons, error messages, and onboarding tours.
- **Tone & Voice:** Defining how the brand "speaks" within the product.
- **Clarity & Logic:** Removing technical jargon and replacing it with plain language.
- **Internationalization (i18n):** Planning how copy will scale across different languages and cultures.

## Tools and Technologies
- **Collaborative Writing:** Figma (editing directly in designs), Google Docs.
- **Content Management:** Contentful, Sanity.io (for headless CMS integration).
- **Localization:** Lokalise, Smartling.
- **Style Guides:** Frontify, Zeroheight.
