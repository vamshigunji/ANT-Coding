# QA Engineer / SDET

## Role
QA (Quality Assurance) Engineers and SDETs (Software Development Engineers in Test) ensure that the software works as intended and is bug-free. SDETs specifically focus on writing code to test code (automation).

## Responsibilities
- **Test Automation:** Writing scripts to automatically test the UI and APIs.
- **Bug Hunting:** Manually exploring the product to find "edge case" failures.
- **Regression Testing:** Ensuring that new features don't break old ones.
- **Performance Testing:** Testing how the app behaves under heavy load.

## Tools and Technologies
- **Automated Testing:** Selenium, Playwright, Cypress, Appium (for mobile).
- **API Testing:** Postman, Rest-Assured.
- **Load Testing:** k6, JMeter, Locust.
- **Test Management:** TestRail, Xray.
- **CI Integration:** Running tests inside GitHub Actions or Jenkins.
