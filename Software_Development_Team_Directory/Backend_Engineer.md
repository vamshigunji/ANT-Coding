# Backend Engineer

## Role
Backend Engineers build the "hidden" logic of the application. They focus on servers, databases, and APIs that power the frontend. Their primary concerns are performance, security, and data integrity.

## Responsibilities
- **API Design:** Creating REST or GraphQL endpoints for the frontend and mobile apps.
- **Database Architecture:** Designing schemas, optimizing queries, and ensuring data is stored correctly.
- **Business Logic:** Implementing the core rules of the application (e.g., payment processing, authentication).
- **Scalability:** Ensuring the backend can handle thousands of concurrent users.

## Tools and Technologies
- **Languages:** Node.js (TypeScript), Python, Go, Java, Rust.
- **Databases:** PostgreSQL, MongoDB, Redis, MySQL.
- **API Standards:** GraphQL (Apollo), REST, gRPC.
- **ORM/Query Builders:** Prisma, Drizzle, TypeORM, SQLAlchemy.
- **Authentication:** Auth0, Clerk, Firebase Auth, JWT.
